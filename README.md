# Mastermind Delphi Game

A Windows Mastermind board game implementation written in **Delphi** (originally Delphi 7 Pro, recompiled in Delphi 12 Community Edition).  
This project demonstrates VCL forms, frames, and object-oriented Delphi programming.

## Features

- Fully playable Mastermind game with **graphical interface**
- Dynamic game board using **TFrame** and **TShape**
- Correct / Nearly markers for guesses
- Optional visual hints for colour-blind users
- Splash screen with logo and short music clip (educational/demo use only)
- Compiles to single executable; no external files required at run-time 

## Notes on Content

## Files Included

- Source code: ".pas", ".dpr", ".dfm", ".dproj"  
- Resources: ".res" files for music and icon  
- Icon file: ".ico"  

## Usage

1. Open the project in Delphi (any version >= Delphi 7).  
2. Compile and run.  
3. Enjoy playing the Mastermind game.
